How this works

  1. 1
    Paste text

    Paste messy text or rich clipboard content into the input box.

  2. 2
    Choose destination

    Select where the cleaned text will be pasted, such as Gmail, Docs, Word, Markdown, Slack, CMS fields, or plain text.

  3. 3
    Review changes

    Use Diff view and the Inspector to see what was changed.

  4. 4
    Copy output

    Use Copy HTML for rich structure and supported styling when available. Use Copy text for plain visible text only; plain-text destinations may only show Copy text.
